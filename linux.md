* 目录进入退出
```
   cd    
   ./    当前目录
   ../   上级目录
   /     根目录
   
   pwd 查看当前路径
   
   ls  查看当前目录下的所有文件、目录
   ls -a 查看隐藏文件
   
   mkdir <name>  在当前目录创建一个文件夹
   mkdir /xxx/xxx  以绝对路径创建 
   touch <name>  在当前目录创建一个文件
   
   vim <name>  进入文件编辑，默认进入时，无法编辑
   a,s,o       进入编辑状态
   Esc         退出编辑状态
   ：          进入文件的命令操作模式
   :q + Enter  退出文件编辑，不保存
   :wq + Enter 保存并退出文件编辑
   :q!         强制退出，不保存
   :wq！        强制保存退出
   dd          进入编辑状态之前，删除一行
   <2> + dd      进入编辑状态之前，删除多行
   
   Ctrl+c      终止当前程序运行
```
